Bantuan / FAQ

Pertanyaan yang Sering Diajukan

Bagaimana cara mengatur ambang batas sensor? ‚ñº

Untuk mengatur ambang batas sensor:

  1. Masuk ke menu "Pengaturan Ambang Batas"
  2. Atur nilai minimal dan maksimal untuk setiap parameter
  3. Klik "Simpan Pengaturan" untuk menyimpan perubahan
Apa yang harus dilakukan jika sensor mengirim data tidak valid? ‚ñº

Jika ada data sensor yang tidak valid:

  1. Periksa menu "Validasi Data Manual"
  2. Pilih data yang bermasalah
  3. Lakukan validasi manual dan berikan catatan
  4. Jika sensor rusak, hubungi teknisi
Bagaimana cara mengunduh laporan harian? ‚ñº

Untuk mengunduh laporan harian:

  1. Buka menu "Laporan Harian"
  2. Pilih tanggal yang diinginkan
  3. Klik "Unduh PDF" untuk mengunduh laporan
Apa arti dari rekomendasi ventilasi? ‚ñº

Rekomendasi ventilasi adalah saran sistem berdasarkan kondisi lingkungan:

  • Buka ventilasi: Suhu atau kelembapan terlalu tinggi
  • Tutup ventilasi: Suhu terlalu rendah
  • Tambah penyiraman: Kelembapan terlalu rendah
  • Tambah/kurangi cahaya: Intensitas tidak optimal
